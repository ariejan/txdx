TxDx uses a plain text todo.txt file. Please unpack the attached ZIP file, which contains
a TxDxTest folder and a todo.txt and archive.txt file. 

 - Start TxDx
 - Click the "Select TxDx folder" button.
 - Navigate to the unzipped `TxDxTest` directory and choose it from the file picker.
 - That's all.
